The Source Autoset  panel allows you to automatically adjust the oscilloscope's vertical and/or horizontal settings for live sources (Ch1...Ch4) to improve measurement accuracy.
Autoset button group:  Select a button to initiate the desired action.
> ALL: Execute both Vertical Scale and then Horizontal Resolution Autoset.
> Vertical Scale: If necessary, change the source's Vertical Scale, Offset and Position so that the signal occupies >=7 vertical divisions.  (Signals with large offsets relative to their peak-peak may end up occupying a smaller vertical range.)
> Horizontal Resolution: Adjust the oscilloscope horizontal resolution (1/sample rate) so that the number of samples on the fastest transition (edge) exceeds a specified target.  When multiple sources are active, the sample rate required by the source with the fastest transitions prevails.   Set the record length to the target value.   (See Optimize Horizontal For button group).
> Undo: Undo the most recent source autoset.  This button is enabled immediately after an autoset is complete.  This button is disabled after a measurement sequence.

Optimize Horizontal For button group:  Select one of the following sample rate / record length targets for Autoset Horizontal Resolution.  The chosen record length will be the shorter of the specified target or the maximum record length available on this oscilloscope.
> Edge Resolution: Minimum sample rate that has >= 5  S/transition.  Target record length is 500k samples.
> Max Edge Count: Minimum sample rate that has >= 2.5 S/transition.  Rj/Dj analysis typically requires a large number of clock cycles or data symbols.  If this oscilloscope does not have a long acquisition memory, it may be necessary trade some edge timing accuracy for a larger data set.  Target record length is 2M samples.
